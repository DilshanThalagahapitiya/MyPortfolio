# Videos Folder

Place your demo videos here!

## Required Videos

Add your video files with these names (or update `scripts/main.js` to match your filenames):

1. `navigation-router.mp4` - Your custom navigation system demo
2. `chat-view.mp4` - Real-time chat interface demo
3. `in-app-purchase.mp4` - StoreKit integration demo
4. `home-screen.mp4` - Home screen with posting/sharing
5. `app-store.mp4` - App Store listing or submission process
6. `testflight.mp4` - TestFlight distribution demo

## Video Specifications

- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 or 1280x720
- **Duration:** 15-30 seconds per video
- **File Size:** Under 50MB each (ideally 5-15MB)
- **Frame Rate:** 30fps

## Quick Recording Tips

1. **Xcode Simulator:** File → Record Screen (Cmd + R)
2. **QuickTime:** File → New Movie Recording → Select iPhone
3. **iOS Device:** Control Center → Screen Recording
